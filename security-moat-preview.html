Approach 01

Static Anatomy

Zero JavaScript. The mark sits beside a numbered list of all six layers. Quietly authoritative — like a textbook diagram on the inside cover of a security whitepaper.

The white core is your IP. Every facet is a layer of defense.

  1. 01

    Auth0 SSO & MFA

    SAML, OAuth 2.0, multi-factor — perimeter authentication on day one.

  2. 02

    Multi-Tenant Row-Level Isolation

    Each org's data fenced off by Postgres RLS at the database layer.

  3. 03

    Step-Up Re-Authentication

    Bulk deletes, status changes, and e-signatures require fresh auth.

  4. 04

    In-Memory Token Storage

    Tokens never touch localStorage. Silent refresh, XSS-resistant.

  5. 05

    Content Sanitization & CSP

    All user content sanitized. Strict CSP, no raw HTML rendering.

  6. 06

    AES-256 Encryption

    HTTPS in transit. Documents in encrypted S3 with presigned URLs.

Approach 03 · Final

Auto-Cycling Showcase

Every 4 seconds the cube highlights one of six defense layers. A node line grows out of the facet ending in the layer name. The matching card below lights up so the reader can read the full detail. Click any facet to jump.

01

AI Intelligence (Zero Retention)

Claude API integration with the zero data retention tier — the feature that attracts prospects first.

02

Auth0 SSO

Enterprise SSO with Auth0 Organizations. SAML and OIDC federation per tenant.

03

RBAC

Four roles (Admin, Attorney, Paralegal, Read-only) plus granular permissions — enforced in your database, not in Auth0.

04

Audit Trails

Immutable, append-only logging on every mutation — the compliance feature pharma firms ask about first.

05

Tenant Isolation

withTenantContext() transaction scoping — no tenant can ever see another's data.

06

Row-Level Security

FORCE ROW LEVEL SECURITY on every table — the bedrock that even a rogue query can't bypass.

Approach 04

Concentric Defense in Depth

A castle-and-moat ring diagram — outermost ring is first line of defense, innermost is closest to the data. The most literal possible interpretation. Honest tradeoff: doesn't use the brand mark, and "defense in depth" rings are a security cliché.

Approach 05

Hybrid Petal Layout

The cube is the centerpiece. Six security cards radiate around it like petals on desktop, stack vertically on mobile. Most ambitious — biggest brand statement, hardest to maintain if the layer count ever changes.
